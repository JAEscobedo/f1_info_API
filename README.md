# F1 Info API

## Project Description
The F1 Info API provides comprehensive data related to Formula 1 racing. It offers a wide range of information including team stats, driver information, and race results.

## Installation Instructions
To install the F1 Info API, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/JAEscobedo/f1_info_API.git
   ```

2. Navigate to the project directory:
   ```bash
   cd f1_info_API
   ```

3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To start using the F1 Info API, run the following command:
```bash
npm start
```
Access the API by navigating to `http://localhost:3000` in your web browser or via a tool like Postman.

## API Endpoints
### /drivers
- **GET**: Retrieve a list of all drivers.

### /teams
- **GET**: Retrieve a list of all teams.

### /races
- **GET**: Retrieve information about upcoming races.
